# 🛠️ QA Productivity & Test-Ops Tools

이 저장소는 QA 프로세스 내 반복적인 수동 작업을 자동화하고, 협업 도구와의 연동을 통해 팀의 생산성을 높이기 위한 도구들을 포함하고 있습니다.

## 📂 Project Structure
- **jira-automation**: JQL 쿼리를 활용한 티켓 조회 및 슬랙/이메일 자동 보고서 발송 스크립트
- **confluence-reporter**: 테스트 결과 및 지표를 Confluence 페이지에 자동으로 업데이트하는 스크립트
- **localization-helper**: 다국어 테스트를 위한 문자열 조합 자동 생성 및 검증 유틸리티

## 🛠 Tech Stack
- **Languages:** Python
- **APIs:** Jira REST API, Confluence REST API, Slack Webhook API
- **Libraries:** Requests, Jira-python, Atlassian-python
